The Random Module

Now that we have our two lists, with each list containing a part of a simile, we need to tell our program to randomly pull an item from each list. We do this by using Python's randommodule.

On lines 6 and 7, we are going to make two new variables. Each variable is going to contain just one item, or value. Type the following code on lines 6 and 7:

part_1 = random.choice(List_1)

part_2 = random.choice(List_2)

The random.choice you see above is randomly pulling an item from whatever is in parentheses next to it and assigning that value to the new variable. So, the variable part_1 will have a value of something pulled from List_1 and the variable part_2 will have a value of something pulled from List_2. This is where the "scrambling" happens--we won't know what values the program pulled until we run the program!
