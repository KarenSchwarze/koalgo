Getting the Final Lines

Let's recap what we've done so far: We've imported Python's random module to allow our program to randomly choose values from our lists of rhyming words to create our couplet. We've created two variables, line_1_rhyming_words and line_2_rhyming_words, each containing a list of words. All of the words rhyme. Next, we created two more variables, line_1 and line_2, and we are asking the user to provide the lines (all but the last word in each line) from the Prologue couplet as the values for these variables.

We have all of the pieces we need for our couplet. Now we just have to somehow get two words and add them to our user input so that we have two complete lines for our couplet.

Because we have imported the random module, we're now able to use it in our program. This module has a function called .choice() that we can use. We can type random.choice(), put the name of a list in the parentheses, and then an item will randomly be chosen from that list.

Let's see how this works. On lines 9 and 10 of your code editor, type:

word = random.choice(line_1_rhyming_words)

print(word)

In the output area below (on the right), you should see one of the words from line_1_rhyming_words. The program randomly chose a word from that list, assigned it as the value of the variable word, and displayed it!

Delete the code on lines 9 and 10. Now that you've seen how random.choice() works, let's use it in our program.

We need to create two more variables. These variables are going to contain the final version of our couplet lines. We will call them line_1_final and line_2_final. Write this on lines 9 and 10 of your code editor:

line_1_final =

line_2_final =

We know that we want to get the final version of each line of our couplet, but we need to teach the program what the formula is for each line. We will use concatenation to add together all of the pieces of each line of the couplet. Concatenation means "adding together". We do this using the + symbol, just as you do when adding in math class.

Before we write the code, let's write it as a formula:

line 1 of couplet = the user input we got for line 1 + a random word from our first list of words

line 2 of couplet = the user input we got for line 2 + a random word from our second list of words

We have variables and values to represent all of these pieces of our formula--let's put them together! Add this code to lines 9 and 10 of your code editor, after the = signs:

line_1 + random.choice(line_1_rhyming_words)

line_2 + random.choice(line_2_rhyming_words)

Now the final version of the first line of our couplet will "equal" the user input (represented by line_1) "plus" a random word from the first list of rhyming words. The final version of the second line of our couplet will "equal" the user input (represented by line_2) "plus" a random word from the second list of rhyming words.
