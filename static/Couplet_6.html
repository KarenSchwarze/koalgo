Your Own Version

Now that you know how to use the random module, how to make variables and values, and how to concatenate, it's time for your to make your own Couplet Creator.

Mapping It Out

Before we write our code in the code editor, it's a good idea to write down our ideas on a piece of paper. That way when we come back to our code editor, we'll have a better idea of what to write.

1. First, we need to find a couplet that we can "revise". Shakespeare wrote a lot of couplets, so you can choose something from one of his plays or poems. You can also search online to find a great couplet.

2. Write down your couplet on your piece of paper, and circle the last words in the lines.

3. Make two lists of words that all rhyme with the last words of the couplet. We want each line to have its own list so that the program does not put the same word at the end of both lines of the couplet.

Writing the Code

Come back to the code editor below. The first thing we need to do is import the randommodule. On line 1, type import random. On lines 3 and 4, write the variables and values for your two lists of words. The format for this is line_1_rhyming_words = ["","","",] and line_2_rhyming_words = ["","",""]with one of your rhyming words within each pair of quotation marks.

On lines 6 and 7, we need to get the user input for each line. Write this code in your editor below:

line_1 = input("Please type the first line of the couplet from the Prologue in Romeo and Juliet, minus the last word. ")

line_2 = input("Please type the second line of the couplet from the Prologue in Romeo and Juliet, minus the last word. ")

On lines 9 and 10, we need to teach our program the formula for each lines. Write this code in your editor:

line_1_final = line_1 + " " + random.choice(line_1_rhyming_words)

line_2_final = line_2 + " " + random.choice(line_2_rhyming_words)

On lines 12 and 13, we need to tell the program to print lines 1 and 2 of our new couplet. Write this code in your editor:

print(line_1_final)

print(line_2_final)

You're done writing your program! Now you just need to click Run as many times as you want to see your couplets. Then click Next to continue.
