Writing the Code

Now that you've learned about indexing in Python, you're ready to write some more code for our oxymoron machine. By now you should have two variables--a list of adjectives and a list of nouns. Now we need to teach our program our oxymoron formula:

oxymoron = adjective + noun

If you think that the formula above kind of looks like a variable followed by values, you're on the right track! We use the = to instantiate, or add values to, a variable. We call the = the assignment operator. It can help to think of it as an equals sign.

On line 3 of our code editor, we need to type the name of our new variable, oxymoron. If we want our oxymoron to be "equal to" the first word in our list of adjectives plus the first word in our list of nouns, what might we need to write as the value of that variable? In other words, what do we need to put on the right side of the equals sign?

If you guessed something like oxymoron = adjectives[0] + nouns[0], you're right!

Before we try out our code, there's one more thing we need to add. If we type in our code just as it is above, the program with put the words "brawling" and "love" right next to each other, without any space between the words. Because we want a space between the words, we can add one by adding quotation marks with a single space in the middle, like this:

oxymoron = adjectives[0] + " " + nouns[0]
