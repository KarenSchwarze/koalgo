Simile Formula

In English, we sometimes have formulas that we can use to construct language. A basic simile follows this formula:

simile = [thing 1] + "is/are/am" + "like/as" + [thing 2]

We're going to teach our program a version of this formula.

On line 9 of your code editor, write this code:

simile = part_1 + " like " + part_2.

To keep things simple, for now we are just using "like", which is one of the possible connection phrases between [thing 1] and [thing 2] in a simile. Other possible connection phrases include like "as", "as though", "as if", "are like", "is like", "is as", and others.

Tip: Make sure to have a space before and after the word like, within the quotation marks, as shown above. Otherwise the words in your simile will be pushed up against each other!

Now we need to tell our program to show our simile in the output area to the right of your editor below. To do this, we can use a print statement. On line 10, write the following code:

print(simile)

Now, when we hit Run, the program will print our simile, which is the formula that we taught it on line 9. Try hitting Run and seeing what happens! Then hit Next.
