Similes are Comparisons

A simile is a comparison of two things. To make our simile generator, we need to have two lists of things that we can compare. On lines 3 and 4, we are going to write two lists.

Our first list is going to be called simply List_1. We're going to take the first part of the similes we showed you earlier and put them in this list. Write the following on line 3 of your code editor (on the left):

List_1 = ["The horse was strutting","There is no frigate","What happens to a dream deferred? Does it dry up","Character is"]

Note that List_1 is called a variable, and each of the items in the list are called values. Think of a variable as a box, and the values as items in the box.

Let's make another list on line 4:

List_2 = ["a prizefighter","a book","a raisin in the sun","a tree and reputation its shadow"]

Now we have two lists of phrases that are going to make up our similes. Click Next to continue.
