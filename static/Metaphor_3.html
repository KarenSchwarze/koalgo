Using the Metaphor Formula

Now that we have our variables and values, we need to teach our program our metaphor formula. Write this code on lines 8-10 of your code editor below:

noun_1 = noun_1a + " is " + noun_1b

noun_2 = noun_2a + " is " + noun_2b

noun_3 = noun_2a + " is " + noun_3b

In this code, we are joining together both nouns in the metaphor with "is". We have a space between the beginning and ending quotation marks around "is" so that we have enough space in our sentence.

In Python, we can use the + symbol to add, or concatenate, pieces of code together.
