Creating Word Lists

First, we need to tell our program that we want to use Python's randommodule. This module will allow the program to randomly choose from the list of ryhming words we will give it so that it can add those words to our couplet. We need to "import", or bring in, the random module, so on line 1 of your code editor (the space on the left below), type:

import random

On line 3 and 4, we are going to make two lists of words. The first list will be the possible end words for the first line of our couplet (including the real end word of the line), and the second list will be the possible end words for the second line of our couplet (including the real end word of the line). Notice that all of the words rhyme. We want to make sure that whatever words the program chooses, we still have a legitimate couplet whose last words rhyme. However, we don't want the program to choose the same word to attach to the end of each line, which is why we need two separate lists.

The names of our lists, line_1_rhyming_words and line_2_rhyming_words, are variables. The words in the list are values. Think of a variable as a box and a value as something you put inside the box. So, line_1_rhyming_words is a variable, and the words in the list are its values.

On line 3, type:

line_1_rhyming_words = ["attend", "mend", "commend"]

On line 4, type:

line_2_rhyming_words = ["end", "suspend", "recommend"]
