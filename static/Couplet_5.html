Seeing the Results

Now that we have taught our program the formula for the lines of our couplet, it's time to tell the program to "print", or display, both lines so that we have a complete couplet.

Write this code on lines 12 and 13 of your code editor:

print(line_1_final)

print(line_2_final)

Click the Run button to see the results!

Try running the program several times to see different versions of your couplet. You may even end up seeing the original couplet from the Prologue--it all depends on which words the program randomly chooses.
