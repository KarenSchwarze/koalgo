Similes

A simile is a kind of figurative language. Figurative language creates pictures in the reader's mind and communicates an idea in a beautiful way.

For this project, you will use similes from literature. But using code, you'll scramble those similes--with entertaining results!

After you have learned how to make a simile generator using these similies, you'll create your very own, original similes in your own version of this project.

Click the Run button to see what you'll make! Run the program several times to see different similies. Then click Next to get started on your first simile generator!
